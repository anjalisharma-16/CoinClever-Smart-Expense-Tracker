# CoinClever-Smart-Expense-Tracker
1. CoinClever is a expense tracker that collect, store, and visually interpret personal expense data to help users understand their spending patterns, track budget utilization, and make informed financial decisions.
2. Each transaction is recorded with description, amount, category, and date. The application analyses this data to generate category-wise breakdowns, monthly spending trends, and budget consumption metrics in real time

# Tools and Technology used:
  I.	HTML&CSS
II.	SheetJS
III.	JavaScript
IV.	JSON
V.	Local Storage
VI.	File Reader API
